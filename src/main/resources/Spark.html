Compreensão do Spark Java:

O Spark Java é um framework web leve e simples para criação de aplicativos web em Java sendo principalmente usado para o desenvolvimento de APIs RESTful.

Comparação com Outros Frameworks:

O Spark é mais simples em suas atriubuições do que o Spring Boot, usado ao longo da disciplina, portanto é útil na criação de aplicativos menores e mais simples

Escalabilidade e Desempenho:

O Spark Java suporta operações assíncronas, o que permite lidar com um grande número de solicitações simultâneas de forma eficiente, garantindo uma boa capacidade de resposta e escalabilidade. Devido à sua arquitetura leve e minimalista, o Spark Java impõe pouca sobrecarga ao sistema, o que torna ele muito mais rápido que as outras APIs.

Facilidade de Uso:

O Spark não necessita de arquivos externos, portanto facilita muito o uso sem tem que se preocupar com várias configurações que podem alterar a depender da aplicação, o que pode atrapalhar o desenvolvimento,
